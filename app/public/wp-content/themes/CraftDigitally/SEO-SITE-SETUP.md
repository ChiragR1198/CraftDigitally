## 🚀 Setup Steps After Theme Activation

### Step 1: Install Essential Plugins
```
Dashboard → Plugins → Add New

Install in this order:
1. Yoast SEO (or Rank Math)
2. W3 Total Cache (or WP Rocket)
3. Wordfence Security
4. UpdraftPlus
```

### Step 2: Configure SEO Plugin
```
SEO → General Settings
- Run configuration wizard
- Connect Google Search Console
- Generate XML sitemap
- Set up social media profiles
```

### Step 3: Setup Caching
```
Performance → General Settings (W3 Total Cache)
- Enable Page Cache
- Enable Browser Cache
- Enable Minification (CSS & JS)
```

### Step 4: Create Pages
```
Pages → Add New

Create these pages:
- Home (use as front page)
- About Us
- Services
- Contact (use "Contact Page" template)
- Blog
- Privacy Policy
- Terms & Conditions
```

Or use quick creator:
```
Appearance → Create Pages → Click "Create Default Pages"
```

### Step 5: Setup Menus
```
Appearance → Menus
- Create "Primary Menu"
- Add: Home, About, Services, Contact, Blog
- Assign to "Primary Menu" location
```

### Step 6: Configure Permalinks
```
Settings → Permalinks
- Select "Post name" (SEO-friendly)
- Save Changes
```

### Step 7: Setup Homepage
```
Settings → Reading
- Select "A static page"
- Homepage: Home
- Posts page: Blog
- Save Changes
```

## 🔧 Troubleshooting

### "Do I need WooCommerce?"
**No!** Only if you want to sell products online. For SEO/Marketing sites, skip it.

### "Which SEO plugin is best?"
**Yoast SEO** for beginners, **Rank Math** for advanced users. Both are excellent.

### "Do I need a page builder?"
**No!** Theme has beautiful built-in templates. But Elementor is great if you want more control.

### "Contact form not working?"
Theme has built-in form. Just use "Contact Page" template when creating contact page.

### "Site is slow?"
Install **WP Rocket** or **W3 Total Cache** and follow performance checklist above.
